# Network Engineering & CCNA Lab Portfolio

## Repository Overview
This repository serves as a practical portfolio of network engineering topologies and configurations built using **Cisco Packet Tracer**. It documents my hands-on progression through enterprise networking concepts, focusing on secure, scalable, and highly available infrastructure design.

These labs are designed to reinforce concepts required for the **Cisco Certified Network Associate (CCNA 200-301)** and general network operations.

### Core Technologies Demonstrated:
* **Routing:** Static Routing, OSPFv2, Inter-VLAN Routing (Router-on-a-Stick).
* **Switching:** VLANs, 802.1Q Trunking, Spanning Tree Protocol (STP), PortFast.
* **Security:** Standard/Extended ACLs, Port Security, SSHv2, DHCP Snooping.
* **Services:** NAT/PAT, DHCP, DNS, NTP, TFTP/FTP configuration management.

## How to Use This Repository
Each lab directory contains:
1. A detailed **`README.md`** explaining the objective, methodology, and proof of configuration.
2. The raw **`.pkt` (Packet Tracer)** file. You can download these files and open them in Cisco Packet Tracer to interact with the simulated hardware, view the running configurations, and test network connectivity yourself.
3. An **`/images`** folder containing traffic analysis and verification screenshots.
